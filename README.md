# Sodiqjonov Abbos Portfolio — Clean Admin

Soddalashtirilgan React + Express portfolio.

## Qolgan asosiy bo‘limlar

- Bosh sahifa
- Men haqimda
- Ko‘nikmalar
- Xizmatlar
- Tajriba
- Loyihalar
- Aloqa
- Yashirin admin panel

## Lokal ishga tushirish

`START.bat` faylini ikki marta bosing. Paket o‘rnatish shart emas.

- Sayt: `http://localhost:3001`
- Admin: `http://localhost:3001/admin`
- Login ma’lumotlari: `ADMIN_LOGIN.txt`

## Admin imkoniyatlari

Profil, loyihalar, xizmatlar, ko‘nikmalar, tajriba, mijoz xabarlari va parolni boshqarish mumkin.

## Terminal intro
Bosh sahifa ochilganda terminal uslubidagi qisqa yuklanish animatsiyasi chiqadi va portfolio avtomatik ochiladi. Admin va loyiha ichki sahifalarida intro ko‘rsatilmaydi.
